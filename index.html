MCP World Agent

ChatGPT에서 내 PC의 CAD, Office, HWP를 바로 호출하세요.

MCP World는 Windows Agent 설치 한 번으로 Word, Excel, PowerPoint, CAD, HWP, Photoshop, Blender, Local Code MCP, OpenCrab Ingest 연결 링크를 발급합니다. 생성된 링크를 ChatGPT 커넥터에 넣으면 파일을 반복 업로드하지 않고 로컬 작업 흐름을 이어갈 수 있습니다.

Why MCP World

파일을 계속 올리고 설명하는 시간을 줄입니다.

Connect

프로그램별 링크 발급

필요한 프로그램을 선택하면 ChatGPT에 붙여 넣을 수 있는 MCP 연결 링크를 바로 확인할 수 있습니다.

Workflow

반복 업로드 감소

도면, 문서, 슬라이드, 코드 폴더 상태를 로컬에서 확인해 매번 파일을 다시 설명하는 과정을 줄입니다.

Control

필요할 때 다시 생성

링크가 노출됐다고 느껴지면 대시보드에서 새 토큰을 발급해 기존 링크를 만료할 수 있습니다.

How it works

설치부터 ChatGPT 연결까지 5단계

  1. 1. 가입Google 또는 이메일로 MCP World 계정을 만듭니다.
  2. 2. 에이전트 설치GitHub Releases에서 Windows 설치 파일을 내려받아 실행합니다.
  3. 3. 도구 선택Word, Excel, PowerPoint, CAD, HWP, Photoshop, Blender, Local Code, OpenCrab 중 필요한 항목을 선택합니다.
  4. 4. 링크 연결대시보드의 MCP 연결 링크를 ChatGPT 커넥터에 붙여 넣습니다.
  5. 5. 관리필요하면 토큰을 다시 생성하거나 연결을 종료해 이전 링크를 닫습니다.

Use Cases

CAD·PPT·HWP 실무자가 반복 설명을 줄이는 흐름에 맞췄습니다.

도면 검토

CAD/ZWCAD 도면의 레이어, 치수, 텍스트 상태를 확인하고 수정 포인트를 정리합니다.

제안서 제작

PowerPoint 슬라이드 구조와 발표 흐름을 점검하고 Excel 표와 함께 다듬습니다.

공문서 자동화

HWP 문서 작성, 서식 점검, 반복 문단 정리를 로컬 파일 우선 방식으로 보조합니다.

Control

연결 링크는 필요할 때 만들고 언제든 다시 생성할 수 있습니다.

원본 파일은 기본적으로 사용자 PC에 두고, 대시보드에서는 앱별 연결 상태와 토큰을 관리합니다. 링크가 노출됐다고 느껴지면 새로 생성해 이전 링크를 더 이상 쓰지 않게 만들 수 있습니다.

  • 앱별 연결 링크 분리
  • 토큰 다시 생성 지원
  • 연결 상태 확인
  • 로컬 파일 우선 작업
  • 설치 파일 GitHub 제공
  • 서비스 고지 문서 제공
