Local MCP SaaS Gateway

ChatGPT에서 내 PC의 CAD, Office, HWP를 바로 호출하세요.

MCP World는 웹 SaaS 계정과 Windows 로컬 에이전트를 연결해 프로그램별 MCP 주소를 발급합니다. 사용자는 파일을 반복 업로드하지 않고, 관리자는 세션과 구독 상태를 한 화면에서 운영할 수 있습니다.

Why this SaaS

브라우저 AI와 로컬 프로그램 사이의 마지막 빈칸을 채웁니다.

Access

로컬 주소 문제

ChatGPT 웹은 사용자 PC의 127.0.0.1 서비스에 직접 접근할 수 없습니다. SaaS가 인증된 세션 주소를 중개해야 합니다.

Workflow

반복 업로드 감소

도면, 문서, 슬라이드 상태를 MCP로 확인하면 매번 파일을 설명하고 다시 올리는 작업을 줄일 수 있습니다.

Business

구독형 운영 기반

사용자, 플랜, 커넥터 발급, 로컬 에이전트 등록 상태를 분리해 실제 SaaS 과금 모델로 확장할 수 있습니다.

Architecture

공개 엔드포인트가 아니라 사용자별 세션 라우트로 연결합니다.

1

ChatGPT Connector

사용자는 프로그램별 MCP 주소를 ChatGPT 커넥터에 등록합니다.

2

SaaS Auth Gateway

계정, 구독 상태, 권한 범위, 세션 만료 시간을 확인합니다.

3

VPS Relay

사용자별 route와 rate limit, 로그 마스킹, tunnel timeout을 적용합니다.

4

Local Agent

Windows PC에서 승인된 로컬 MCP 서버만 실행하고 상태를 보고합니다.

Workflow

가입부터 ChatGPT 연결까지 5단계

  1. 1. 가입Google 또는 이메일 계정으로 SaaS 워크스페이스를 만듭니다.
  2. 2. 에이전트 설치Windows 전용 MCPWorld 에이전트를 내려받아 실행합니다.
  3. 3. 프로그램 등록Office, CAD, HWP, Photoshop, Blender 실행 파일을 등록합니다.
  4. 4. 주소 복사프로그램별 MCP 세션 주소를 ChatGPT 커넥터에 붙여 넣습니다.
  5. 5. 세션 종료작업 후 세션을 만료해 외부 접근 경로를 닫습니다.

Use Cases

CAD·PPT·HWP 실무자가 반복 설명을 줄이는 흐름에 맞췄습니다.

도면 검토

CAD/ZWCAD 도면의 레이어, 치수, 텍스트 상태를 확인하고 수정 포인트를 정리합니다.

제안서 제작

PowerPoint 슬라이드 구조와 발표 흐름을 점검하고 Excel 표와 함께 다듬습니다.

공문서 자동화

HWP 문서 작성, 서식 점검, 반복 문단 정리를 로컬 파일 우선 방식으로 보조합니다.

Security Model

공개 URL 하나로 모든 프로그램을 열어두지 않습니다.

각 커넥터는 사용자 계정, 플랜, 세션 만료, 로컬 에이전트 등록 상태를 확인한 뒤 발급됩니다. 원본 파일은 기본적으로 사용자 PC에 머물고, SaaS는 연결 권한과 세션 라우팅만 관리하는 구조를 목표로 합니다.

  • 사용자별 단기 세션 주소
  • 프로그램별 커넥터 분리
  • 구독 상태 기반 접근 제어
  • 관리자 이벤트 로그
  • 로컬 파일 원본 저장 최소화
  • 정책 문서와 고지 화면 분리
  • 결제정보 직접 저장 금지
  • 로컬 경로와 세션 키 마스킹

Production Deployment

지금 폴더 그대로 정적 사이트로 올릴 수 있습니다.

VPS에서는 Nginx 또는 Caddy의 웹 루트에 정적 파일을 배포하고, 백엔드 API와 Cloudflare Worker 프록시를 함께 연결해 랜딩, 사용자 대시보드, 관리자 콘솔을 운영합니다.
