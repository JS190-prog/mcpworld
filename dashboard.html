에이전트 다운로드 GitHub Repo 설치 안내
  1. 에이전트를 다운로드해 설치합니다.
  2. 필요한 프로그램의 에이전트 등록 버튼을 눌러 로컬 앱과 연결합니다.
  3. 주소 복사를 눌러 ChatGPT MCP 커넥터에 붙여 넣습니다.

Program Connectors

프로그램별 MCP 주소

각 프로그램은 별도 세션 주소를 사용합니다. 등록된 프로그램만 ChatGPT에 붙여 넣을 주소를 복사할 수 있게 구성했습니다.

다운로드 후 바로 연동 에이전트를 설치한 뒤 각 프로그램의 에이전트 등록 버튼을 누르면 mcpworld://pair 링크가 열립니다. 토큰 다시 생성을 누르면 기존 MCP 주소는 즉시 만료되고 새 주소만 사용할 수 있습니다.
