# 동의 화면 UI 계획

작성일: 2026-06-18 KST

## 회원가입 동의

필수 동의와 선택 동의를 분리한다.

필수:
- 서비스 이용약관
- 개인정보 처리방침

선택:
- 마케팅 안내 수신
- 서비스 소식 수신

선택 항목에 동의하지 않아도 핵심 서비스 이용은 가능해야 한다.

## 소셜 로그인 안내

소셜 계정으로 시작하는 버튼 아래에는 서비스 이용약관과 개인정보 처리방침 링크를 둔다.

수집 정보는 로그인에 필요한 최소 범위로 제한한다.

## 쿠키 설정

쿠키 설정은 다음 범주로 나눈다.

- 필수: 로그인 유지와 보안 목적
- 기능: 언어, 테마, 화면 설정
- 분석: 방문 통계와 기능 사용량
- 선택 홍보: 기본값은 꺼짐

## 로컬 에이전트 안내

설치 전 별도 안내 화면을 둔다.

확인 항목:

1. 설치 목적
2. 연결되는 로컬 프로그램 범위
3. 수행 가능한 작업 범위
4. 원본 파일 변경 가능성
5. 복사본 작업 권장
6. 연결 종료 방법
7. 제거 방법

## 결제 전 확인

결제 전 요금, 결제 주기, 자동 갱신, 환불 정책, 해지 방법을 요약해서 보여준다.
