MCP World 문의 폼 구성

Google Form 생성 권한이 준비되면 아래 항목으로 폼을 만들고, 공유 링크를 site-config.js의 supportForm 값에 넣으면 메인 사이트의 문의 버튼이 바로 연결됩니다.

자동 생성을 원하면 Google Apps Script 생성 스크립트를 새 Apps Script 프로젝트에 붙여 넣고 실행하세요.

폼 제목

MCP World 설치/연결 문의

폼 설명

GitHub 계정이 없어도 MCP World 설치, 로그인, Agent 연결, 도구 호출 문제를 보낼 수 있는 문의 폼입니다.
