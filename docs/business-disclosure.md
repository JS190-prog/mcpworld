# 사업자 정보 및 통신판매 고지 초안

작성일: 2026-06-18 KST

## 목적

유료 SaaS 결제 전 사용자가 사업자와 거래조건을 확인할 수 있도록 표시할 정보를 정리한다.

## 표시 항목

- 상호
- 대표자 성명
- 사업장 주소
- 전화번호
- 이메일
- 사업자등록번호
- 통신판매업 신고번호
- 신고 기관
- 호스팅 서비스 제공자
- 고객지원 연락처

## 거래조건 고지 항목

- 서비스 명칭과 내용
- 요금제별 가격
- 결제 방법
- 결제 주기
- 서비스 제공 방식
- 구독 해지 방법
- 환불 및 청약철회 조건
- 장애와 분쟁 처리 방법
- 약관 확인 방법

## 결제 전 확인 모달

결제 버튼 앞에 다음 정보를 요약한다.

1. 선택한 요금제
2. 결제 금액
3. 결제 주기
4. 다음 결제 예정일
5. 구독 취소 방법
6. 환불 정책 링크
7. 자동결제 동의 문구

## 가격 변경 또는 무료에서 유료 전환

정기결제 금액이 인상되거나 무료 제공이 유료 결제로 전환되는 경우에는 사전에 변경 시점, 변경 전후 가격, 결제 방법, 해지 방법을 고지하고 동의를 받는다.
