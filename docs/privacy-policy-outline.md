# 개인정보 처리방침 작성 개요

작성일: 2026-06-18 KST

## 목적

MCP World SaaS의 회원, 결제, 로컬 에이전트, 서비스 이용 기록 처리 기준을 정리한다.

## 필수 구성

1. 처리 목적
2. 처리 항목
3. 보유 기간
4. 파기 절차와 방법
5. 이용자와 법정대리인의 권리 및 행사 방법
6. 보호책임자와 문의처
7. 안전성 확보 조치
8. 처리방침 변경 고지
9. 열람, 정정, 삭제, 처리정지 요구 절차
10. 열람청구 접수 부서
11. 권익침해 구제 방법

## 항목 분류

| 구분 | 예시 항목 | 목적 |
|---|---|---|
| 일반 가입 | 이메일, 닉네임, 계정 보호값 | 계정 생성과 로그인 |
| Google 로그인 | Google 식별값, 이메일, 표시 이름 | 소셜 로그인 |
| 결제 | 결제사 고객 식별자, 구독 상태 | 유료 서비스 제공 |
| 로컬 에이전트 | 기기명, 연결 상태, 최근 접속 시각 | MCP 세션 관리 |
| 자동 수집 | 접속 기록, 보안 로그, 쿠키 | 보안과 장애 대응 |

## 조건부 고지

- 전송요구권 적용 여부
- 자동화된 결정 사용 여부
- 처리위탁 업체
- 국외 이전 여부
- 업로드 파일 제공 시 별도 보관 기간

## 반영 원칙

로컬 원본 파일은 기본적으로 서버에 보관하지 않는다. 서버에 보관되는 자료는 필요한 범위로 최소화하고 보호 조치를 적용한다.
