로컬 에이전트 및 MCP 사용 고지 초안

작성일: 2026-06-18 KST

목적

사용자가 로컬 에이전트와 MCP 연결의 작동 범위와 위험을 이해한 뒤 서비스를 이용하도록 안내한다.

핵심 고지

이 서비스는 사용자의 PC에서 실행 중인 프로그램을 MCP를 통해 연결할 수 있다.

대상 예시:

  • CAD
  • PowerPoint
  • Excel
  • HWP
  • Photoshop
  • Blender

사용자 확인 사항

서비스 이용 전 사용자는 다음을 확인해야 한다.

  • 본인이 권한을 가진 파일과 프로그램만 연결한다.
  • 중요한 파일은 복사본으로 작업한다.
  • 원본 파일 변경 가능성을 이해한다.
  • 작업 결과는 사용자가 최종 확인한다.
  • 필요 시 즉시 세션을 종료한다.

서비스 설계 원칙

  • 원본 덮어쓰기 전 확인 절차를 우선한다.
  • 위험 작업은 미리보기 또는 사용자 확인을 거친다.
  • 연결은 사용자별 세션으로 분리한다.
  • 작업 종료 후 세션 종료 버튼을 제공한다.
  • 서버에는 로컬 파일 원문을 기본 보관하지 않는다.

책임 문구 방향

회사는 AI와 MCP 자동화 결과의 완전성을 보장하지 않는다. 다만 회사의 고의 또는 중대한 과실로 인한 책임은 관계 법령에 따른다.
